Projects

Valor

A game released on the Roblox platform in late 2022, Valor is a medieval team-based action game. It features large scale AI battles and seperate player controlled gameplay. I codeveloped this game with user Brick_man and a large part of this project was both the grid based map system and the pathfinding needed for the npcs to navigate that system. Since the map was generated uniquely we had to use a custom A* pathfinding algorithm.
